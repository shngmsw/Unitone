# Privacy Policy / プライバシーポリシー

## 日本語版

### はじめに

本プライバシーポリシーは、Unitone（以下「本アプリ」）におけるユーザー情報の取り扱いについて説明します。

### 収集する情報

本アプリは以下の情報を**ローカルにのみ**保存します：

1. **サービス設定情報**
   - 追加されたチャットサービスのURL
   - サービスの表示順序
   - ウィンドウサイズ・位置の設定

2. **認証情報**
   - 各チャットサービスのログインセッション（Cookieを含む）
   - これらはElectronのwebviewセッションとして端末内に保存されます

### 収集しない情報

本アプリは以下の情報を**収集・送信しません**：

- チャットメッセージの内容
- 連絡先情報
- 使用統計・分析データ
- 位置情報
- 個人を特定できる情報（PII）

### データの保存場所

すべてのデータはユーザーの端末内にのみ保存されます：

- **Windows**: `%APPDATA%/unitone/`
- **macOS**: `~/Library/Application Support/unitone/`
- **Linux**: `~/.config/unitone/`

### 第三者サービス

本アプリは以下の第三者サービスへのアクセスを提供します：

- Slack (https://app.slack.com)
- Google Chat (https://chat.google.com)
- Microsoft Teams (https://teams.microsoft.com)
- Chatwork (https://www.chatwork.com)
- ユーザーが追加したカスタムサービス

これらのサービスの利用には、各サービスのプライバシーポリシーが適用されます。

### AIコンパニオン機能

AIコンパニオン機能（Gemini）を使用する場合：
- ユーザーが明示的に入力した内容のみがAPIに送信されます
- 会話履歴はローカルに保存されます
- Google のプライバシーポリシーが適用されます

### データの削除

アプリをアンインストールすると、すべてのローカルデータが削除されます。手動でデータを削除する場合は、上記の保存場所のフォルダを削除してください。

### 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。

### ポリシーの変更

本ポリシーを変更する場合は、アプリのアップデートを通じて通知します。

### お問い合わせ

プライバシーに関するご質問は、GitHubリポジトリのIssueまたは以下までご連絡ください：
- GitHub: https://github.com/shngmsw/Unitone

## English Version

### Introduction

This Privacy Policy describes how Unitone ("the App") handles user information.

### Information We Collect

The App stores the following information **locally only**:

1. **Service Configuration**
   - URLs of added chat services
   - Service display order
   - Window size and position settings

2. **Authentication Data**
   - Login sessions for each chat service (including cookies)
   - These are stored on your device as Electron webview sessions

### Information We Do NOT Collect

The App does **NOT collect or transmit**:

- Chat message contents
- Contact information
- Usage statistics or analytics
- Location data
- Personally Identifiable Information (PII)

### Data Storage Location

All data is stored exclusively on the user's device:

- **Windows**: `%APPDATA%/unitone/`
- **macOS**: `~/Library/Application Support/unitone/`
- **Linux**: `~/.config/unitone/`

### Third-Party Services

The App provides access to the following third-party services:

- Slack (https://app.slack.com)
- Google Chat (https://chat.google.com)
- Microsoft Teams (https://teams.microsoft.com)
- Chatwork (https://www.chatwork.com)
- Custom services added by the user

Use of these services is subject to their respective privacy policies.

### AI Companion Feature

When using the AI Companion feature (Gemini):
- Only content explicitly entered by the user is sent to the API
- Conversation history is stored locally
- Google's Privacy Policy applies

### Data Deletion

Uninstalling the App will delete all local data. To manually delete data, remove the folders at the storage locations listed above.

### Children's Privacy

This App is not intended for children under 13 years of age.

### Changes to This Policy

We will notify users of any changes to this policy through app updates.

### Contact Us

For privacy-related questions, please contact us via GitHub Issues or:
- GitHub: https://github.com/shngmsw/Unitone
