# Store Description / ストア説明文

**App Name / アプリ名**: Unitone
**Category / カテゴリ**: Productivity / 仕事効率化

## 日本語版

### 短い説明（80文字以内）

複数のチャットツールを1つに。Slack・Teams・Google Chat・Chatworkをまとめて管理。

### 詳細説明

**すべてのチャットを、ひとつに。**

Unitoneは、ビジネスで使う複数のチャットツールを1つのウィンドウで管理できるデスクトップアプリです。もう複数のブラウザタブを開いたり、アプリを切り替えたりする必要はありません。

#### 主な機能

**複数サービスを一元管理**
- Slack、Google Chat、Microsoft Teams、Chatworkに対応
- カスタムサービスも追加可能 - 社内チャットやその他のWebサービスも統合

**使いやすいインターフェース**
- サイドバーでサービスを素早く切り替え
- ドラッグ&ドロップでサービスの順序をカスタマイズ
- 通知バッジで未読メッセージを一目で確認

**生産性を向上**
- 1つのウィンドウでチャットを集中管理
- システムトレイに常駐してすぐにアクセス
- AIコンパニオン機能で作業をサポート（Gemini搭載）

**クロスプラットフォーム**
- Windows、macOS、Linuxに対応
- どのOSでも同じ体験

#### こんな方におすすめ

- 複数のプロジェクトで異なるチャットツールを使い分けている方
- チャットアプリの切り替えにストレスを感じている方
- チャットの通知を一元管理したい方
- リモートワークで複数のコミュニケーションツールを使用している方

#### 注意事項

- 各チャットサービスのアカウントが別途必要です
- インターネット接続が必要です
- 本アプリは各チャットサービスの非公式クライアントです

## English Version

### Short Description (within 80 characters)

All your chats in one place. Manage Slack, Teams, Google Chat & Chatwork together.

### Full Description

**All your chats, unified.**

Unitone is a desktop app that lets you manage multiple business chat tools in a single window. No more switching between browser tabs or jumping between apps.

#### Key Features

**Unified Chat Management**
- Support for Slack, Google Chat, Microsoft Teams, and Chatwork
- Add custom services - integrate internal chat systems and other web services

**Intuitive Interface**
- Quickly switch between services using the sidebar
- Customize service order with drag & drop
- See unread messages at a glance with notification badges

**Boost Your Productivity**
- Manage all chats in one focused window
- Stay accessible from the system tray
- AI companion feature to assist your work (powered by Gemini)

**Cross-Platform**
- Available for Windows, macOS, and Linux
- Consistent experience across all operating systems

#### Perfect For

- Professionals using different chat tools across multiple projects
- Anyone tired of constantly switching between chat apps
- Users who want centralized notification management
- Remote workers using multiple communication tools

#### Notes

- Separate accounts for each chat service are required
- Internet connection required
- This app is an unofficial client for each chat service
